# Bin Chicken Rare Biosphere Genomes Notes

- The Zenodo concept DOI is `10.5281/zenodo.14889982`.
- The dataset is supplementary data for the Bin Chicken targeted metagenomic coassembly preprint, associated with the code repository `https://github.com/AroneyS/binchicken/`.
- The Zenodo description reports 77,562 microbial genomes recovered from public metagenomes through 800 sample-group coassemblies.
- The current latest Zenodo record is `15220963`, version `Revised metadata`, published on 2025-04-15.
- The apparent size reduction from roughly 30 GB to roughly 900 MB is a versioning issue, not a compressed replacement for the same MAG archive. The latest version contains only `binchicken_RBGs_metadata.tar.gz`.
- The terrestrial/engineered/host MAG archive remains available at explicit record `14915155`, version `terrestrial_engineered_host_mags`, with file `binchicken_RBGs_terrestrial_engineered_host.tar.gz`.
- The aquatic MAG archive is another explicit version in the same concept record family: `14890002`, version `aquatic_mags`, with file `binchicken_RBGs_aquatic.tar.gz`.
- The original metadata record is `14889983`, version `metadata`; it has been superseded by the revised metadata record.
- For reproducible downloads, pin record IDs instead of resolving only the concept DOI, because Zenodo's latest-version behavior currently hides the MAG archives from the latest landing page.
