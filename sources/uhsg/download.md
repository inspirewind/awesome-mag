# UHSG Download Notes

UHSG data are available through CNSA project `CNP0002131`. Use the public metadata directory first:

- CNGBdb project: `https://db.cngb.org/data_resources/project/CNP0002131`
- public metadata directory: `https://ftp.cngb.org/pub/CNSA/data7/public_info/CNP0002131/`
- assembly manifest: `https://ftp.cngb.org/pub/CNSA/data7/public_info/CNP0002131/data_download_links_CNP0002131_ftp.txt`
- assembly metadata: `https://ftp.cngb.org/pub/CNSA/data7/public_info/CNP0002131/metadata_CNP0002131_assembly.tsv`
- experiment metadata: `https://ftp.cngb.org/pub/CNSA/data7/public_info/CNP0002131/metadata_CNP0002131_experiment.tsv`
- sample metadata: `https://ftp.cngb.org/pub/CNSA/data7/public_info/CNP0002131/metadata_CNP0002131_sample_Microbial_sample.tsv`

## Metadata Files

Download the small metadata files first:

```bash
wget -c https://ftp.cngb.org/pub/CNSA/data7/public_info/CNP0002131/data_download_links_CNP0002131_ftp.txt
wget -c https://ftp.cngb.org/pub/CNSA/data7/public_info/CNP0002131/metadata_CNP0002131_assembly.tsv
wget -c https://ftp.cngb.org/pub/CNSA/data7/public_info/CNP0002131/metadata_CNP0002131_experiment.tsv
wget -c https://ftp.cngb.org/pub/CNSA/data7/public_info/CNP0002131/metadata_CNP0002131_sample_Microbial_sample.tsv
```

`metadata_CNP0002131_assembly.tsv` contains the assembly accession, FASTA file name, MD5 checksum, assembly method, sequencing technology, total size, contig count, and N50 for each MAG.

## Genome FASTA Files

The manifest contains per-assembly URLs with this pattern:

```text
assembly ftp://ftp.cngb.org/pub/CNSA/data5/CNP0002131/<sample_accession>/<assembly_accession>/<fasta_file_name>
```

Example:

```text
assembly ftp://ftp.cngb.org/pub/CNSA/data5/CNP0002131/CNS0410152/CNA0032467/1111525849480213_bin.6.fa.gz
```

Extract the assembly URLs and download them with `wget`:

```bash
awk '$1 == "assembly" {print $2}' data_download_links_CNP0002131_ftp.txt > uhsg_assembly_ftp.urls
wget -c -i uhsg_assembly_ftp.urls
```

If `ftp.cngb.org` is slow from your network, replace the host with `ftp2.cngb.org`:

```bash
sed 's#ftp://ftp.cngb.org/#ftp://ftp2.cngb.org/#' uhsg_assembly_ftp.urls > uhsg_assembly_ftp2.urls
wget -c -i uhsg_assembly_ftp2.urls
```

## Raw Read Metadata

The experiment metadata records WGS runs and paired FASTQ names. It is useful when mapping the MAGs back to raw sequencing data:

- `experiment_accession`
- `run_accession`
- `library_strategy`
- `platform`
- `instrument_model`
- `file_name` / `file_md5`
- `file2_name` / `file2_md5`

## Verification

Use the assembly metadata checksums after download:

```bash
awk -F'\t' 'NR > 1 {print $15 "  " $14}' metadata_CNP0002131_assembly.tsv > uhsg_assembly_md5.txt
md5sum -c uhsg_assembly_md5.txt
```

The checksum command assumes all downloaded FASTA files are in the current directory. If files are kept in accession subdirectories, rewrite the second column using the paths generated by the manifest.

## Automation Decision

No source-specific script is needed right now. CNSA already provides a public manifest, metadata TSV files, and MD5 checksums. A future helper would only need to filter the manifest by sample or accession and then call `wget` or `aria2c`.
