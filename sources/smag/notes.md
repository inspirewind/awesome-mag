# SMAG Notes

SMAG is a soil-focused MAG catalog from the Global Soil MAGs Project. The project reports 40,039 metagenome-assembled genome bins reconstructed from 3,304 soil metagenomes across 9 ecosystems, clustered into 21,077 species-level genome bins. Of those SGBs, 16,530 are reported as unknown SGBs.

The resource is in scope for Awesome MAG because it is an original project portal with a public Zenodo dataset, companion analysis code, and publication-backed soil MAG catalog. It belongs most naturally under soil and terrestrial environment MAG resources.

The project page also links a SMAG web interface, but the page currently labels it as under website maintenance. The GitHub repository contains pipeline and figure scripts rather than a packaged browser/download tool.

Primary citation:

- Ma, B., Lu, C., Wang, Y. et al. A genomic catalogue of soil microbiomes boosts mining of biodiversity and genetic resources. Nature Communications 14, 7318 (2023). https://doi.org/10.1038/s41467-023-43000-z
